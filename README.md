# bs-monad

This project provides the implementation of basic monads in ReasonML.

The following monads are available so far:

1. Writer Monad
2. Reader Monad
3. State Monad
4. List Monad
5. Option Monad
6. Promise Monad
7. Continuation Monad

# Build
```
npm run build
```

# Build + Watch

```
npm run start
```


# Editor
If you use `vscode`, Press `Windows + Shift + B` it will build automatically
